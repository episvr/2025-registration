# 2025 Fall Registration

一个社团招新注册墙的物理模拟可视化展示。

它同时也是一个 Git 练手项目，旨在帮助新成员熟悉 **Fork**、**Commit** 和 **Pull Request** 的开源协作流程。通过提交 PR，可以将自己的头像和简介加入到这个炫酷（？）的物理世界中！

点击头像可以查看个人信息，并且可拖拽社团成员的头像并发生碰撞。

## 🛠️ 本地开发与构建

```shell
# 1. 安装项目依赖
npm install

# 2. 启动本地开发服务器 (实时预览修改)
npm run dev

# 3. 构建生产版本 (用于部署)
npm run build

# 4. 预览构建产物
npm run preview
```

## 🚀 快速开始

1. Fork 本仓库
2. 填写你的个人信息到 `data/members/`
3. 提交 Pull Request (PR)
4. 等待合并，查看你的头像出现在墙上
